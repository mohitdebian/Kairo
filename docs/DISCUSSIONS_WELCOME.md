## 🌌 What is Kairo?

Kairo is an open-source browser that rethinks how you interact with the web. Instead of drowning in tabs, Kairo organizes your digital life into **Spaces**, **Folders**, and **AI-powered Workflows** — giving you a browser that works the way your brain does.

Built with **Electron**, **React**, and a passion for design, Kairo is crafted for power users, developers, and researchers who demand more from their tools.

## 🚀 Getting Started

New here? Here's how to jump in:

1. ⭐ **Star the repo** — It helps us grow and motivates the team.
2. 📖 **Read the [README](https://github.com/mohitdebian/Kairo#readme)** — Understand the vision, features, and architecture.
3. 🛠️ **Set up locally** — Clone, install, and run Kairo in under 2 minutes:
   ```bash
   git clone https://github.com/mohitdebian/Kairo.git
   cd Kairo && npm install && npm run dev
   ```
4. 🤝 **Read our [Contributing Guide](https://github.com/mohitdebian/Kairo/blob/main/CONTRIBUTING.md)** — Learn how to submit your first PR.
5. 💬 **Introduce yourself below!** — Tell us who you are and what excites you about Kairo.

## 🛣️ What's on the Roadmap?

We're building Kairo in phases:

- **Phase 1** — Core Browser (Tabs, Spaces, Navigation, History) — _In Progress_ 🔨
- **Phase 2** — Productivity Features (Split View, Command Palette, Folders)
- **Phase 3** — AI Features (Smart Grouping, Page Summarization)
- **Phase 4** — Sync & Collaboration (E2E Encrypted Sync)
- **Phase 5** — Ecosystem (Extensions API)

Check the full [ROADMAP.md](https://github.com/mohitdebian/Kairo/blob/main/ROADMAP.md) for details.
