# Funding and Sustainability

Open-source browsers are notoriously expensive and time-consuming to build. To ensure Kairo can be maintained long-term, we plan to implement the following funding strategies once we hit Phase 2 of our Roadmap.

## 1. GitHub Sponsors
We will set up a GitHub Sponsors profile for the core organization.
- **Tiers:** Range from $5/mo (Community Backer) to $500/mo (Corporate Sponsor).
- **Perks:** Corporate sponsors will get their logo in the Kairo `README.md` and a special shoutout in our Release Notes.

## 2. Open Collective
For transparent expense tracking (server costs for sync, domain names, signing certificates).
- Anyone can donate.
- The community can see exactly where every dollar is spent.

## 3. Future Monetization (Optional)
Kairo's core browser will **always** be free and open-source. However, if backend infrastructure is required (like hosting LLMs for AI features or servers for End-to-End Encrypted Sync), we may introduce a "Kairo Pro" tier. This allows users to pay for hosted sync and AI usage, while keeping the client code 100% open source (so you can self-host if you prefer!).
